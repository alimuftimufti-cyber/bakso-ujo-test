
# PANDUAN CARA ONLINE (BAKSO UJO)

Halo Boss! Ikuti langkah ini pelan-pelan ya.

## TAHAP 1: SIAPKAN AKUN (Cukup Sekali Seumur Hidup)
1. Buka [GitHub.com](https://github.com) -> Daftar Akun (Gratis).
2. Buka [Vercel.com](https://vercel.com) -> Login pakai akun GitHub tadi.

## TAHAP 2: UPLOAD KODE (Setiap ada update)
1. Di GitHub, buat "New Repository" (Gudang Baru), kasih nama `bakso-ujo`.
2. Upload semua file kodingan ini ke sana.
3. Klik "Commit Changes".

## TAHAP 3: ONLINE-KAN (DEPLOY)
1. Di Vercel, klik "Add New Project".
2. Pilih `bakso-ujo`.
3. Klik "Deploy".
4. Tunggu sebentar, nanti akan muncul Link Website (misal: bakso-ujo.vercel.app).
5. Link itu sudah bisa dibuka di HP kasir, HP dapur, dan HP owner!

## CATATAN PENTING (Versi 1)
Saat ini aplikasi menggunakan penyimpanan **LOKAL (HP Masing-masing)**.
Artinya:
- Data di HP Kasir HANYA ada di HP Kasir.
- Data di HP Owner HANYA ada di HP Owner.
- Mereka belum saling "ngobrol".

Jika ingin data saling terhubung (Real-time), kabari saya:
*"Saya sudah berhasil deploy Versi 1, tolong buatkan Backend Database untuk Versi 2"*.
